B · connection setup Two separate invoicing connections behind the same setupCard + StructuredConfigSection + CredentialsPanel slots. To connect KSeF you only need an environment + auth (token) — that authorises your KSeF mailbox. The seller details print on invoices (the issuer block) and aren’t fetched from KSeF, so they’re collected separately and needed before your first issue. Subiekt is an invoicing-only connection (your store/warehouse stays PrestaShop, Allegro, …) and gets its seller details from Subiekt itself.
