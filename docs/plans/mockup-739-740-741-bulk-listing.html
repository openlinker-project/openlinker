Implementation notes

Every pixel uses var(--…) tokens — no hardcoded colors, no rem leaks. Status pills carry mono+caps + leading dot per the StatusBadge contract (color is never the only signal). DataTable rows 36px; KPI cards 96px; buttons 28/32/38px; page content starts ≤120px from top in the actual viewport. Sidebar 240px, top bar 52px — both the cockpit-frame contract.

BulkActionBar is the only new shared/ui primitive this PR adds — it fulfils the style-guide promise where docs/frontend-ui-style-guide.md § Core Component Patterns already listed it.
